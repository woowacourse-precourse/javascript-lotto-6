## 기능 구현 목록

### 로또 구입 금액 입력받기

- 문구: 구입금액을 입력해 주세요.

### 로또 구입 금액 유효한지 확인

- 입력을 받았는지
- 숫자가 아닌 경우
- 1000원으로 나누어 떨어지지 않는 경우
- throw 발생

### 발행한 로또 수량 출력

- 문구: n개를 구매했습니다.
- 입력받은 금액에 해당하는 만큼 로또 발행<br>
  (1장당 1000원)

### 발행한 로또 번호 출력

- 문구: [n1, n2, n3, n4, n5, n6]
- 오름차순으로 출력

### 당첨 번호 입력받기

- 문구: 당첨 번호를 입력해 주세요.
- 6개 입력
- 쉼표(,) 기준으로 구분

### 당첨 번호 유효한지 확인

- 입력을 받았는지
- 숫자가 아닌 경우
- 번호가 6개가 아닌 경우
- 중복된 숫자 있는 경우
- 숫자 범위가 1~45를 벗어난 경우
- throw 발생

### 보너스 번호 입력받기

- 문구: 보너스 번호를 입력해 주세요.
- 1개 입력

### 보너스 번호 유효한지 확인

- 입력을 받았는지
- 숫자가 아닌 경우
- 번호가 1개가 아닌 경우
- 당첨번호와 중복되는 경우
- 숫자 범위가 1~45를 벗어난 경우
- throw 발생

### 사용자가 구매한 로또 번호와 당첨 번호 비교

- 1등: 6개 번호 일치 / 2,000,000,000원
- 2등: 5개 번호 + 보너스 번호 일치 / 30,000,000원
- 3등: 5개 번호 일치 / 1,500,000원
- 4등: 4개 번호 일치 / 50,000원
- 5등: 3개 번호 일치 / 5,000원

### 당첨 내역 출력

- 문구: 당첨 통계<br>
  3개 일치 (5,000원) - n개<br>
  4개 일치 (50,000원) - n개<br>
  5개 일치 (1,500,000원) - n개<br>
  5개 일치, 보너스 볼 일치 (30,000,000원) - n개<br>
  6개 일치 (2,000,000,000원) - n개

### 수익률 출력

- 문구: 총 수익률은 n%입니다.
- 소수점 둘째 자리에서 반올림
