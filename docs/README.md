# 🎯 구현 조건

함수(또는 메서드)의 길이가 15라인을 넘어가지 않도록 구현한다 - 함수는 한 가지 일을 하도록 기능을 쪼갠다

indent depth가 3이 넘지 않도록 구현한다. 2까지만 허용한다

기능단위로 커밋한다

Jest를 이용하여 기능 목록이 정상 동작함을 테스트 코드로 확인한다 - 도메인 로직에 단위 테스트를 구현한다

else를 지양한다
