## 기능 구현 목록
---

1. 로또 게임
    - [ ] 로또 금액을 입력하면 그에 해당하는 로또를 발행하기
    - [ ] 당첨 번호와 보너스 번호 입력 받기
    - [ ] 로또 번호를 오름차순으로 정렬
    - [ ] 사용자가 구매한 로또 번호와 당첨 번호를 비교하기
    - [ ] 당첨 내역과 수익률을 출력하기


2. 예외 처리
    - [ ] 입력한 로또 금액이 천원 단위인지 확인
    - [ ] 입력한 로또 금액이 숫자인지 확인
    - [ ] 입력 받은 당첨 번호가 쉼표로 구분되는지 확인
    - [ ] 입력 번호의 로또 범위가 1~45 사이인지 확인
    - [ ] 로또 번호를 발행할 때, 중복되는지 확인
    - [ ] 보너스 번호가 6개의 로또 번호와 중복되는지 확인
