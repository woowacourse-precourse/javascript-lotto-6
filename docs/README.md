### 입력 받기
- 구입 금액
    - 1000원 당 한장
        - 1000원으로 나누어 떨어짐 ✅
    - 한장 당 6개의 랜덤 숫자
        - 오름차순 정렬 ✅
        - 겹치지 않게 검증 ✅
- 당첨 번호
    - 쉼표를 기준으로 구분 ✅
    - 겹치는 수 여부 검증
- 보너스 번호
    - 당첨 번호와 겹침 여부 검증
### 결과 출력
- 당첨 내역
    - 각 로또와 당첨 번호 비교
    - 겹칠 때마다 count
    - 보너스 번호가 겹치면 다르게 처리
- 수익률
    - (당첨 금액 / 구입 금액) * 100
    - 소수점 둘째 자리에서 반올림

### 예외 처리
- 잘못된 값 입력
- 당첨 번호, 보너스 번호가 겹치는 경우
