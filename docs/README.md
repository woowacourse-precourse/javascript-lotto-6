# MEMO

- 사람이 로또기계를 통해 로또를 사고 로또가게에서 결과를 확인.
- 사람
  - 하는 일 : 로또사기, 로또가게 가기
  - 가지고 있는것 : 돈, 로또목록
- 로또
  - 하는 일 : 생성되기
  - 가지고 있는것 : 번호
- 로또를 발행하는 기계
  - 하는 일 : 생성하기
- 로또가게 :
  - 하는 일 : 당첨번호 받기, 당첨여부확인하기, 수익률 계산하기.
  - 가지고 있는 것 : 당첨번호, 보너스 번호, 구매결과
