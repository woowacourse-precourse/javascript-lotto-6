## 로또 게임
### 1️⃣ 게임 방법
#### 1. 로또 구입하기
- 구입 금액을 입력한다. 로또 개당 금액은 1000원이며 입력한 금액만큼 로또가 구매된다.
#### 2. 로또 번호 받기
- 구입한 로또의 개수 만큼 개당 6개의 숫자와 1개의 보너스 숫자를 입력한다.
#### 3. 당첨 결과 확인하기
- 번호가 일치하는 만큼 다음과 같은 당첨금을 획득한다.
- 1등: 6개 번호 일치 / 2,000,000,000원
- 2등: 5개 번호 + 보너스 번호 일치 / 30,000,000원
- 3등: 5개 번호 일치 / 1,500,000원
- 4등: 4개 번호 일치 / 50,000원
- 5등: 3개 번호 일치 / 5,000원
