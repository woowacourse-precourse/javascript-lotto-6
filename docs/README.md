로또 프로그램
=============
## 기능 목록
- 로또 구입 금액을 입력
- 구입 금액 예외 처리
    - 문자열을 입력한 경우
    - 천원 단위가 아닌 경우
- 로또 발행
    - 로또 구입 금액에 맞춰 발행/로또 1장 당 1000원
    - 중복되지 않는 6개의 숫자
- 당첨 번호 입력
    - 쉼표를 기준으로 구분
- 보너스 번호 입력
- 당첨 번호와 보너스 번호 입력 예외 처리
    - 숫자가 중복된 경우
    - 1~45 사이의 정수가 아닌 경우
    - 당첨 번호에 보너스 번호가 포함된 경우
- 당첨 번호 비교
- 당첨 내역 출력
- 수익률 출력
    - 수익률은 소수점 둘째 자리에서 반올림
