# 우테코 프리코스 3주차 기능 목록

## 로또

#### 개요

로또 게임을 구현한다.

#### 기능

[x] 로또 구입 금액을 입력받는다.

    [x] 1,000원 단위가 아니면 예외를 발생시킨다.

    [x] 숫자 형식이 아니면 예외를 발생시킨다.

[x] 발행한 로또 수량과 번호를 출력한다.

    [x] 로또 번호는 오름차순으로 정렬하여 출력한다.

[x] 당첨 번호와 보너스 번호를 입력 받는다.

    [x] 번호는 , 기준으로 구분한다.

    [x] 번호의 범위는 1 에서 45 사이의 숫자여야 한다.

[x] 당첨 내역을 출력한다.

[x] 수익률을 출력한다.

    [x] 수익률은 소수점 둘째 자리에서 반올림 한다.
