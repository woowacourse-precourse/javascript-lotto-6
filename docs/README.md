# 기능 구현 목록

## 금액 입력
 - [x] 금액을 입력받는다
 - [x] 금액의 유효성을 검사한다
 - [x] 금액의 /1000 만큼 반환한다

## 당첨 번호 입력
 - [x] 당첨 번호를 입력받는다
 - [x] 숫자를 나눈다
 - [x] 숫자가 담긴 리스트들을 유효성검사를한다
 - [x] 당첨 번호를 반환한다

 ## 보너스번호 입력
 - [x] 보너스 번호를 입력받는다
 - [x] 보너스 번호 유효성 검사를한다
 - [x] 보너스번호를 반환한다

## 로또 발행 및 출력
 - [x] 로또를 발행한다
 - [x] 로또가 발행될때마다 출력한다
 - [x] 발행된 로또를 리스트에 담는다
 - [x] 모두 발행되면 로또리스트를 반환한다

## 로또 결과 출력
 - [x] 개인 로또 결과 객체를 생성한다
 - [x] 로또 리스트의 당첨번호, 보너스번호의 일치수를 대조한다
 - [x] 객체의 로또 등수별 당첨수와 총 이익을 출력한다
