## 로또게임 실행하기

- 패키지 설치를 위해 Node.js 버전 18.17.1 이상이 필요하다.
```
npm install
```
- 설치 후 다음 명령어를 통해 테스트 코드를 실행할 수 있습니다.
```
npm test
```
- 혹은 node 환경에서 실제 로또를 실행해볼 수 있습니다.
```
node ./src/index.js
```

## 구현시 지켜야 할 개인적인 약속
- 하나의 기능을 구현하면 해당하는 테스트 코드를 작성합니다.
