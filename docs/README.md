# 구현 기능 목록
- 구매 금액 입력
  - 1000원 단위
  - 숫자
- 로또 번호 출력
  - 구매 금액을 1000원으로 나누어 발행할 로또의 수량
  - 각 로또는 1~45 사이의 서로다른 6개의 랜덤값
  - 로또 번호는 오름차순으로 정렬
- 당첨 번호 입력
  - 쉼표를 기준으로 구분
  - 여섯개의 숫자
  - 1에서 45 사이의 숫자
  - 중복되는지 확인
- 보너스 번호 입력
  - 1에서 45 사이의 숫자
- 당첨 통계 출력
  - 몇 개가 일치하는지 계산
  - 수익금과 로또 구입 금액을 통해 수익률을 계산하여 출력
