# 기능 구현 목록

## LottoController Class

- [x] 사용자로부터 금액을 입력 받는다.
- [x] 금액 예외 처리.

  ### Counter Class

  - [x] 입력받은 금액으로 살 수 있는 로또 개수 결정.
  - [x] 개수만큼 로또 생성.

- [x] 생성한 로또 번호 출력.

  ### WinningLottoMachine Class

  - [x] 당첨 번호와 보너스 번호 입력 받기.
  - [x] 입력 번호 예외 처리.

  ### LottoComparison Class

  - [x] 각 로또 번호를 당첨 번호와 보너스 번호를 비교.
  - [x] 비교한 결과와 수익률을 계산하여 출력.
