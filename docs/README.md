# 기능 목록

<br>

## 입력

- ### 로또 구입 금액 입력

  > **체크 리스트**

  - [x] 1000원 단위로 나눠 떨어지는지 체크

  - [x] 숫자형 체크

  - [x] 정수 체크

  - [x] 양수 체크

  <br>

  > **정상 케이스**

  ```bash
    12000
  ```

  <br>

  > **예외 케이스**

  ```bash
    1200
    [ERROR] 1000원 단위가 아닙니다.

    asdf
    [ERROR] 숫자 형식이 아닙니다.

    1000.3
    [ERROR] 정수가 아닙니다.

    -1000
    [ERROR] 앙의 정수가 아닙니다.
  ```

- ### 당첨 번호 입력

  > **체크 리스트**

  - [x] 숫자형 체크

  - [x] 정수 체크

  - [x] 당첨 번호 개수 및 쉼표 구분자 체크

  - [x] 1~45 범위의 자연수 입력 체크

  - [x] 번호끼리 중복 체크

  <br>

  > **정상 케이스**

  ```bash
    1,2,3,4,5,6
  ```

  <br>

  > **예외 케이스**

  ```bash
    asdf
    [ERROR] 6개의 번호가 아니거나 ","로 구분하지 않았습니다

    1,2,3,4,5,6,7
    [ERROR] 6개의 번호가 아니거나 ","로 구분하지 않았습니다

    1,2,3?4/5,6.
    [ERROR] 6개의 번호가 아니거나 ","로 구분하지 않았습니다

    1,2,3,4,5,12.5
    [ERROR] 정수가 아닙니다.

    0,1,2,3,4,5
    [ERROR] 양의 정수가 아닙니다.

    1,2,3,4,5,49
    [ERROR] 각 번호가 1~45 사이 자연수이지 않습니다.

    1,2,3,4,5,5
    [ERROR] 중복된 번호가 존재합니다.
  ```

- ### [x] 보너스 번호 입력

  > **체크 리스트**

  - [x] 숫자형 체크

  - [x] 정수 체크

  - [x] 1~45 범위의 자연수 입력 체크

  - [x] 당첨 번호와 중복 여부 체크

  <br>

  > **정상 케이스**

  ```bash
    33
  ```

  <br>

  > **예외 케이스**

  ```bash
    asdf
    [ERROR] 숫자 형식이 아닙니다.

    12.5
    [ERROR] 정수가 아닙니다.

    -10
    [양의 정수가 아닙니다.]

    50
    [ERROR] 각 번호가 1~45 사이 자연수이지 않습니다.

    1        // 당첨 번호에 1이 있을 때.
    [ERROR] 당첨 번호와 중복됩니다.
  ```

## 출력

- ### 발행한 로또 수량 및 번호 출력

  > **체크 리스트**

  - [x] 로또 번호 오름차순 정렬

  - [x] 당첨 내역 출력

  <br>

  > **정상 케이스**

  ```bash
    2개를 구매했습니다.
    [14, 20, 29, 31, 32, 43]
    [1, 2, 14, 15, 17, 32]
  ```

- ### 당첨 출력

  > **체크 리스트**

  - [x] 일치하는 개수 비교

  <br>

  > **정상 케이스**

  ```bash
    당첨 통계
    ---
    3개 일치 (5,000원) - 1개
    4개 일치 (50,000원) - 0개
    5개 일치 (1,500,000원) - 3개
    5개 일치, 보너스 볼 일치 (30,000,000원) - 0개
    6개 일치 (2,000,000,000원) - 0개
  ```

- ### 수익률 출력

  > **체크 리스트**

  - [x] 소수점 둘째 자리에서 반올림

  <br>

  > **정상 케이스**

  ```bash
    총 수익률은 10.0%입니다.
  ```

## 로또 발행

- ### 로또 발행마다 중복되지 않는 6개 숫자 뽑고 정렬

  > **체크 리스트**

  - [x] 숫자가 6개가 아닌 경우 예외 발생

  - [x] 중복되는 경우 예외 발생

  - [x] Random.pickUniqueNumbersInRange 사용

  <br>

  > **예외 케이스**

  ```base
            // 로또 번호가 6개가 아닐 때
    [ERROR] 로또 번호는 6개여야 합니다.

            // 로또 변호들 중 중복된 번호가 있을 때
    [ERROR] 중복된 번호가 존재합니다.
  ```

## 당첨 체크

- ### 로또 번호와 당첨 번호를 비교해 일치하는 번호 개수 카운트

  > **체크 리스트**

  - [x] 보너스 볼 번호는 로또 번호 중 5개가 일치할 때만 사용

## 다시 입력

- ### 잘못된 값 입력 시 예외 발생 후 해당 부분부터 다시 입력

  > **체크 리스트**

  - [x] 각 입력 단계 성공 체크 플래그 설정

  <br>

  > **정상 케이스**

  ```base
    보너스 번호를 입력해 주세요.
    50
    [ERROR] 각 번호가 1~45 사이 자연수이지 않습니다.


    보너스 번호를 입력해 주세요.
    6
  ```

# 기능 테스트

<br>

## 핵심 도메인 로직

- ### Lotto

  - [x] 로또 발행마다 6개 숫자 정렬

- ### Lottery

  - [x] 구입 금액에 해당하는 만큼 로또 발행

  - [x] 로또 번호와 당첨 번호를 비교해 일치하는 번호 개수 확인

    - [x] 당첨 번호와 일치하는 번호 개수 확인

    - [x] 5개 번호 일치할 때만 보너스 번호 사용
