# 기능 명세

## 도메인

도메인에서는 다음과 같은 기능을 구현합니다.

- [ ] 구입 금액에 대한 유효성 검사
- [ ] 구입 금액만큼 로또 발행
- [ ] 당첨 번호에 대한 유효성 검사
- [ ] 보너스 번호에 대한 유효성 검사
- [ ] 로또 추첨
- [ ] 1등부터 5등까지 당첨 수량 분석
- [ ] 수익률 분석

## UI

UI에서는 다음과 같은 기능을 구현합니다.

- [ ] 구입 금액 입력
- [ ] 발행 로또 수량 및 번호 출력
- [ ] 당첨 번호 입력
- [ ] 보너스 번호 입력
- [ ] 당첨 통계 출력
- [ ] 수익률 출력
